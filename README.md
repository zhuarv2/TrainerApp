# Trainer App
     
A Progressive Web App (PWA) that helps users manage and remember their daily workout schedule.

## Features (Planned)

- User registration and login
- Weekly workout planner
- Daily workout reminder
- Edit/Delete workout schedule
- Progressive Web App support

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI
- Database: PostgreSQL
- Deployment: Oracle Cloud Free Tier
